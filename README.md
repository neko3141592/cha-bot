## 開発

このBOTの開発に機能を追加したい場合は、以下の手順に従ってください。

1. フォークします。
2. 新しいブランチを作成します。
   ```bash
   git checkout -b feature/YourFeature
   ```
3. 変更をコミットします。
   ```bash
   git commit -m "Add your feature"
   ```
4. プッシュします。
   ```bash
   git push origin feature/YourFeature
   ```
5. プルリクエストを作成します。
